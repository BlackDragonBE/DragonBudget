# Budgets

## What budgets do

A budget sets a monthly spending limit for a category. Once set, the Dashboard and Budgets pages show a progress bar for each category: green while you're within the limit, red once you go over.

Budgets are **per category, per month** — you set them independently for each month, which means you can adjust limits seasonally.

## Setting a budget

1. Go to the [Budgets](/budgets) page
2. Select the month using the month picker at the top
3. Click the amount next to any expense category to edit it inline
4. Type the monthly limit in euros and press Enter (or click away)

Setting a limit to **€0** removes the budget for that category.

Income categories appear in a separate section on the right. Instead of a spending limit, you enter the amount you **expect to receive** — the app then shows actual received vs expected.

## Copying from a previous month

Rather than entering limits from scratch each month, click **"Copy from previous month"** to duplicate the previous month's budget limits into the current month. Existing limits are not overwritten — only categories with no limit yet are filled in.

This is useful at the start of a new month when your limits haven't changed much.

## Reading the Budgets page

Each expense category row shows:

| Element | Meaning |
|---------|---------|
| Progress bar | Green = within budget, red = over budget |
| Left number | Amount spent so far this month |
| Right number | Your monthly limit (or "available" for sinking fund categories) |
| Percentage | Spent ÷ limit |

The **total row** at the bottom aggregates all expense categories.

If any transactions remain uncategorized, their total appears below the category list as a reminder.

## Budgets and sinking funds

Categories with the sinking fund (rollover) feature enabled show an **available** amount instead of the raw limit. Available = limit + any amount carried over from previous months. See [Sinking Funds](/help#sinking-funds) for the full explanation.

## Budgets in the Dashboard

The Dashboard's spending section mirrors the Budgets page for the selected month. Click any category name to jump directly to that category's transactions in the Transactions page.
