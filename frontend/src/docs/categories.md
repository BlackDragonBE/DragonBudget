# Categories

## What categories are for

Every transaction can be assigned to exactly one category. Categories let you group spending (Groceries, Rent, Dining out…) and income (Salary, Freelance…) so you can track where your money comes from and goes.

## Creating a category

On the [Categories](/categories) page, fill in the form at the top:

- **Icon** — any emoji, up to 16 characters. Shown everywhere the category appears.
- **Name** — a short label like "Groceries" or "Utilities".

After creating a category you can edit its colour and other settings inline in the list below.

## Category settings

Each category row in the list has these editable fields:

| Field | Description |
|-------|-------------|
| **Icon** | Emoji displayed next to the category name |
| **Name** | Category label |
| **Colour** | Used in charts and progress bars on the Dashboard and Budgets pages |
| **Income** | Check this if the category represents money coming in (salary, refunds, etc.) |

Income and expense categories are kept separate throughout the app — income shows in its own section on the Dashboard and Budgets pages and is never counted against your spending totals.

## Archiving vs deleting

**Archive** hides a category from the active list and from dropdowns without removing it from history. Use this for categories you no longer need but that still have old transactions attached. Archived categories appear at 50% opacity; click "Restore" to bring one back.

**Delete** permanently removes a category. This is only possible if no transactions are currently assigned to it. Deleting a category also removes all its associated rules and budgets.

## Transactions without a category

Transactions that haven't been categorized yet are called "uncategorized." The Dashboard shows a dedicated widget listing all uncategorized transactions for the current month so you can categorize them quickly via the inline dropdown.

## Tips

- Use the colour picker to assign distinct colours to your top spending categories — the stacked bar charts on the All-time page become much easier to read.
- If you have a catch-all category like "Miscellaneous", give it a neutral grey colour so it doesn't dominate the charts.
- Categories marked as Income don't need a budget limit — they show on the Budgets page as expected vs received amounts instead.
